# 计算机科学词汇表

这里蒐集了计算机科学里，常用词汇之译法。

# 为什么需要词汇表？

* 译者翻译需要一个统一的参考

* 菜鸟学习需要知道术语的中文

* 作为读 CS 书的辞典使用亦可

# 如何使用

dict.md 包含了 A-Z 所有的词汇。

可使用字母索引表浏览或浏览器搜寻功能 (CMD+F, CTRL+F)。

# 如何贡献

欢迎贡献！集思广益！

发送 Pull Request 提交词汇表中尚未出现的词汇，请按照相同格式编排。

发现词汇有错误或更好的译法请开 [Issue][issue] 讨论。

靠大家的力量来一起壮大这个词汇表。
